# FlippyCat
Its not flappy bird. I swear!

## Running
To start the game, run `./gradlew :run`

## Features
 - A sensible tutorial
 - Somewhat readable buttons
 - A playable game
 - An auto-solver that can be enabled in the `Constants.java` file
